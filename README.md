# Gestor de Domótica para Hogares

Aplicación de escritorio desarrollada en C++ con Qt5 que simula el control de dispositivos domóticos como luces, persianas y sensores.

## Tecnologías

- C++17
- Qt5 (Widgets)
- SQLite3
- Doxygen
- Git / GitHub

## Funcionalidades

- Registrar dispositivos con nombre y tipo
- Cambiar estado: Encendido, Apagado o Fallo
- Persistencia de datos en base de datos local
- Registro de eventos con fecha y hora
- Historial de eventos en tabla interactiva

## Cómo compilar

```bash
cd GestorDomotica
qmake GestorDomotica.pro
make
./GestorDomotica
```
